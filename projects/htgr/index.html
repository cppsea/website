About The Project

  • Our HTGR project aims to provide people an intro into machine learning. Our HTGR team have utilized Random Forest, SVM, And LSTM models.
  • Random forest models use random subsets of data labels and random feature sets to create multiple decision trees. Each tree is branched through various conditions, which might involve comparing data set values like mean acceleration, to certain thresholds.
  • SVM models use a hyperplane of data, using the confidence level of the data point to create a binary classification scenario, which with high confidence could be very useful. LSTM models use sequential data passed through memory cells to decide which information to forget and which to save as it moves to the next cell. Using these 3 models, a broad yet highly accurate range was cast by splitting the data up into 3 groups. Their future plans include continuation of gathering more data and uploading documentation on the accuracy.
