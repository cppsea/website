CS Catalog

A functioning website people will utilize in order to properly gauge certain skills and advancements ahead of time. The primary goal of this project is to facilitate greater understanding and willingness to learn within the field of machine learning for students who may be confused on how to begin.

About The Project

  • At CS Catalog, our team created an API to provide another way into machine learning and create a website which tracks and logs users progress. Divided into teams, one group focused on Figma and React to create the UI/UX. The API team used Node.js and Express to create their API. The database team utilized Railway and Postgres.
  • CS Catalog serves as a specified GPT, allowing users to delve into topics on machine learning. In a collaborative environment, you are able to agree or disagree with articles within the text, allowing for the site to continually push those articles that are generally agreed upon. CS Catalog contains an ever-expanding database that users will be able to utilize for information. They plan to provide better security in the future, allowing for sign in and authentication encryption.
